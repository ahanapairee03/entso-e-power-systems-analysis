# ENTSO-E Power Systems Analysis

MSc Dissertation – University College London  
Author: Ahana Pairee  

## Project Overview
This project analyses European electricity system data from ENTSO-E using dimensionality reduction techniques such as PCA, autoencoders, and NNMF.

The aim is to identify patterns in interconnected power systems across Europe.

## Data Source
ENTSO-E Transparency Platform

## Structure
- notebooks/ → exploratory analysis  
- src/ → data processing scripts  
- data/ → sample datasets  
- results/ → figures and outputs
